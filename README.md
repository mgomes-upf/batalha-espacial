# batalha-espacial
Projeto acadêmico da disciplina Computação Gráfica e Realidade Virtual, ministrada pelo professor Rafael Rieder na Universidade de Passo Fundo (UPF). O trabalho desenvolve um jogo de Batalha Espacial em Realidade Virtual (VR), onde o jogador pilota uma nave, destrói inimigos e evita obstáculos dentro de um tempo limite.

## Equipe
- Eduardo Steffens Hoppen, 198272
- Leonardo Cologneze Dias, 178873
- Mateus Gomes, 198175

## Descrição
- Jogo em primeira pessoa com nave espacial em VR.
- Objetivo: destruir inimigos e chegar ao destino dentro do tempo.
- Dois níveis de dificuldade: Fácil e Difícil.
