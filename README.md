# Batalha Espacial

Projeto desenvolvido para o Trabalho Prático da disciplina de *Computação Gráfica e Realidade Virtual, ministrada pelo **Prof. Dr. Rafael Rieder* na *Universidade de Passo Fundo*.

O jogo foi construído em *Unity 2021.3.45f1*, utilizando elementos 2D, colisões, animações, música, efeitos sonoros e um sistema simples de pontuação, dificuldade e telas de menu.

---

## Sobre

O jogo "Batalha Espacial" é um jogo de nave em 2D no qual o jogador deve sobreviver ao ataque de asteroides, coletar pontos e tentar alcançar a maior pontuação possível antes de ser destruído.

O projeto contém:

- Menu Principal  
- Sistema de Dificuldade (Fácil e Difícil)  
- Tela Sobre  
- Tela de Vitória  
- Tela de Game Over  
- Pontuação e HighScore  
- Obstáculos (Asteroides)  
- Efeitos sonoros, animações e explosões  

---

## Como Jogar

- Utilize *W/A/S/D* ou as *setas* para movimentar a nave  
- Use *clique esquerdo* para atirar  
- Desvie dos asteroides  
- Elimine inimigos para ganhar pontuação  
- Alcançando pontuação suficiente → Vitória  
- Perdendo toda a vida → Game Over
